# Korean Menu Viewer

This web application displays Korean food menus fetched from an external API. It provides a user-friendly interface to browse and filter menu options.

## Key Features

*   **Dynamic Menu Loading:** Fetches and displays menu data from a remote API.
*   **Filtering:** Filter menus by workplace, date, and meal category (breakfast, lunch, dinner, etc.).
*   **Geolocation:** Automatically finds the nearest workplace based on the user's current location.
*   **Responsive Design:** The layout is optimized for both desktop and mobile devices.

## How to Use

1.  Open the `index.html` file in a web browser.
2.  Use the dropdown menu at the top to select a workplace.
3.  Use the date picker to choose a specific date.
4.  Click on the category buttons (e.g., "조식" for breakfast, "중식" for lunch) to filter the menu.
5.  Click the "현위치" (Current Location) button to automatically select the nearest workplace.
